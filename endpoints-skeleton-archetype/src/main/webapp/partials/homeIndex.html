Scheduler App

Web based Schedulling application, provides both client side for business clients to book appointments.An Admin side for Businesses to administer appoinments as well as many other built in feautures.

Client Features:

  • Mobile Friendly
  • Book and View Appointments
  • Interative Calendar
  • Cancel Appointments

Admin Features:

  • Full Access of Calendar
  • Appointment Modification
  • Interactive Calendar
  • Client Account Administration
  • Account Security Clearances
  • Email Reminders
